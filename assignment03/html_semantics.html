HTML <code> Tag

Overview

The <code>tag represents a fragment of computer code â€” for example, a snippet of HTML or CSS, or some other programming language code.

This is an inline tag.

By default the content of <code> element is displayed in a monospace (fixed-width) font, such as "Courier", but this style can be overridden using the CSS font-family property.

HTML <details> Tag

Overview

The <details> tag specifies additional details that the user can open and close on demand.

This tag is often used to create an interactive widget that the user can open and close. By default, the widget is closed. When open, it expands, and displays the content within.

Ths is a block tag.

Syntax

The basic syntax of the <details> tag:

<details>...</details>

NOTE: The <summary> tag is used in conjuction with <details> to specify a visible heading for the details.

Example

<details>
    <p>HTML is a markup language for describing the structure of web pages.</p>
</details>

This is how it is displayed on the page:

HTML is a markup language for describing the structure of web pages.

HTML <summary> Tag

Overview

The <summary> tag defines a visible heading for the <details> element. The heading can be clicked to view/hide the details.

The <summary> element should be the first child element of the <details> element.

This is a block tag.

Syntax

The basic syntax of the <summary> tag:

<summary>...</summary>

Example

<details>
    <summary>What is HTML?</summary>
    <p>HTML is a markup language for describing the structure of web pages.</p>
</details>

This is how it is displayed on the page:

What is HTML?

HTML is a markup language for describing the structure of web pages.
