CSS "list-style-type" Property

Overview

This CSS property allows you to set different list item markers for ordered or unordered lists.

There is a wide variaty of values that can be set: none, disc, circle, square, decimal, decimal-leading-zero, armenian, georgian, lower-alpha, upper-alpha, lower-greek, lower-latin, upper-latin, lower-roman, upper-roman, inherit.

Syntax

ul {
    list-style-type: circle;
}

Example

CSS "border-radius" Property

Overview

The border-radius property defines the radius of the element's corners. This property allows you to add rounded corners to elements.

This property can have from one to four values. Here are the rules

  • Four values - border-radius: 15px 50px 30px 5px:

    1. top-left corner
    2. top-right corner
    3. bottom-right corner
    4. bottom-left corner

  • Three values - border-radius: 15px 50px 30px:

    1. top-left corner
    2. top-right corner and bottom-left corners
    3. bottom-right corner

  • Two values - border-radius: 15px 50px:

    1. top-left and bottom-right corners
    2. top-right and bottom-left corners

  • One value - border-radius: 15px:

    1. the value applies to all four corners, which are rounded equally

Syntax

img {
    border-radius: 50%;
}

CSS "opacity" Property

Overview

The opacity property sets the opacity level for an element.

The opacity-level describes the transparency-level, where 1 is not transparent at all, 0.5 is 50% see-through, and 0 is completely transparent.

Values of the opacity property:

  • number - Specifies the opacity. From 0.0 to 1.0
  • initial - Sets this property to its default value (1.0).
  • inherit - Inherits this property from its parent element.

Syntax

img {
    opacity: 0.6;
}
